Design notes — Mockup C

  • Cards give an immediate visual summary — no need to read through a table
  • Improvement badge (▲ +2' 3") replaces the need to mentally compare first vs. best mark
  • All-time rank badge in the card header, not buried in a table column
  • Mini sparkline per card gives shape-of-career at a glance
  • Could replace the current PR table, or live alongside it (cards above, table below for detail)
  • Relay cards gracefully show "not enough data for chart" without breaking the layout
  • Downside: takes more vertical space than the current clean table layout
  • Downside: cards need thoughtful mobile layout (currently 2-col, could be 1-col on mobile)
