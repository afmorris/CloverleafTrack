Design notes — Mockup B

  • Self-contained section — doesn't touch any existing UI
  • Tabs make it easy to navigate multi-event athletes
  • Summary stats (competitions, improvement, rank) tell the story without reading the chart
  • The "PR at time" orange dots show the progression of PRs even in a losing performance
  • Most valuable when athlete has 2+ seasons — single-season athletes see same data as Mockup A
  • Downside: a separate section means scrolling past the existing performance tables
